## 현재 코드 상태
- PhotoItem.swift: Identifiable, Codable compatible model (assetIdentifier 저장, resolveAsset() 메서드로 PHAsset 복구)
- AlbumManager.swift: 앨범 생성/삭제/이동, albumPhotos 매핑
- DataManager.swift: UserDefaults 저장/로드
- ContentView.swift: 일부 연동 완료, ImageLoader가 PHAsset? 받음

## 수정해야 할 항목들

### 1. ContentView.swift 수정
a) ImageLoader의 asset 파라미터 타입을 PHAsset?으로 변경하고, photos[currentIndex].resolveAsset()을 전달하도록 수정
b) onSwipeUp에서 AlbumManager.shared.movePhoto(toAlbum:albumName:photoId:)를 호출하도록 수정
c) 앨범 내보내기 버튼 추가
   - 현재 선택된 앨범의 모든 사진을 PHAsset에서 UIImage로 변환
   - UIImage를jpeg Data로 변환
   - 임시 디렉토리에 ZIP 파일 생성
   - ShareSheet로 다른 앱으로 공유

### 2. ImageLoader.swift 생성 또는 수정
- ImageLoader가 PHAsset?을 받도록 하고, resolveAsset()을 통해 PHAsset을 로드
- PHAsset이 nil일 경우(이미지 로딩 실패 또는 macOS 시뮬레이터 아님) placeholder 이미지 표시

### 3. 앨범 내보내기 기능 구현
- ContentView에 "앨범 내보내기" 버튼 추가
- 현재 선택된 앨범의 모든 사진들을 PHAsset에서 UIImage로 변환
- UIImage를 jpeg Data(80% 품질)로 변환
- ZIP 파일로 묶기: FileManager로 임시 디렉토리를 만들고 각 사진을 photo_0.jpg, photo_1.jpg 등으로 저장 후 ZIP 압축
- ShareSheet로 내보내기: URL을 ShareSheet에 전달

### 4. UI 개선
- 사진 위 더블탭(gesture로 .onTapGesture(count: 2))으로 선택 체크마크 표시
- 앨범 변경 시 토스트 알림을 상단 배너 스타일로 변경 (toastMessage -> bannerMessage, position 변경)
- 빈 화면일 때 안내 메시지 더 구체화

### 5. AlbumManager 수정
- album에 속한 사진 ID 목록(albumPhotos)을 album move 시 업데이트
- 앨범 삭제 시 albumPhotos에서 해당 앨범 제거
