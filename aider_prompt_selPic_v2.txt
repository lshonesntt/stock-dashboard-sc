## 중요한 수정 사항들

### 1. Photo 모델 수정 (가장 시급)
현재 Photo는 PHAsset을 포함하고 있어서 Codable이 될 수 없어. 수정 방향:
- Photo.id만 유지하고 asset은 transient (non-Codable) 프로퍼티로 변경
- assetIdentifier를 저장하고, 뷰에서 PHAsset으로 복구하는 로직 추가
- 또는 PHAsset 식별자 배열을 저장하고, 표시할 때 PHAsset.fetchAssets로 가져오기

정확한 수정 방법:
```swift
struct PhotoItem: Identifiable, Codable {
    let id: UUID
    let assetIdentifier: String  // PHAsset.localIdentifier
    var album: String?
    transient var asset: PHAsset?  // 디스패치 스레드에서 복구
    
    init(asset: PHAsset?) {
        self.id = UUID()
        if let asset = asset {
            self.assetIdentifier = asset.localIdentifier
            self.asset = asset
        } else {
            self.assetIdentifier = ""
            self.asset = nil
        }
        self.album = nil
    }
    
    init(id: UUID, assetIdentifier: String, album: String?) {
        self.id = id
        self.assetIdentifier = assetIdentifier
        self.album = album
        self.asset = nil
    }
    
    func resolveAsset() -> PHAsset? {
        if let asset = asset { return asset }
        guard !assetIdentifier.isEmpty else { return nil }
        let result = PHAsset.fetchAssets(withLocalIdentifiers: [assetIdentifier], options: nil)
        return result.firstObject
    }
}
```

### 2. DataManager 수정
- PhotoItem으로 변경
- UserDefaults에 저장할 때 assetIdentifier만 저장 (PHAsset은 인코딩 불가)
- 로드 시 PHAsset으로 복원

### 3. AlbumManager 수정
- albumPhotos: [String: [UUID]] (앨범 이름 -> 사진 ID 목록)
- 앨범 이동 시 albumPhotos에도 업데이트
- 앨범 삭제 시 해당 앨범에 속한 사진의 album 속성도 제거하고 albumPhotos에서 제거

### 4. macOS 호환성
- #if targetEnvironment(simulator) 분기 사용하여 macOS simulator에서는 Photos 프레임워크 사용
- macOS 시뮬레이터가 아닌 경우 (real macOS)에는 앱 내 사진 추가 UI를 대체로 사용
- 또는 macOS 시뮬레이터에서만 Photos 기능 활성화, 그 외에는 제한된 기능

### 5. 앨범 내보내기 기능 구현
- 앨범 선택 버튼 추가 (앨범별 필터링)
- 선택된 앨범의 모든 PHAsset을 UIImage로 변환
- UIImage를 Data(jpeg 80%)로 변환
- ZIP 파일 생성 (PHPicker나 ShareSheetsheet를 통해 다른 앱으로 공유)
- ShareSheet로 내보내기 구현

### 6. UI 개선
- 사진 위 더블탭으로 선택 체크마크 표시 (UI로 사진이 선택된 사진 목록 표시하는건 아님)
- 앨범 변경 시 상단 배너 스타일 피드백 (toastMessage -> bannerMessage)
- 빈 화면일 때 안내 메시지 구체화 (이미 구현됨)
- 앨범 내보내기 버튼 추가

### 7. 이미지 전환 애니메이션
- Image 전환 시 .transition(.move(edge: .trailing)) 또는 .blurEffect 적용
