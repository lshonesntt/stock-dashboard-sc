# Windows용 IP분류도구

## 설치 방법

### 방법 1: 빌드 버전 (PyInstaller)
1. `build.bat` 더블클릭
2. `dist/IP분류도구/IP분류도구.exe` 실행

### 방법 2: Python 버전 (직접 실행)
1. Python 3.9 이상 설치
2. `pip install PyQt5 openpyxl xlrd`
3. `python app.py` 실행

## 사용 방법
1. IP분류도구.exe 실행
2. 원본 Excel 파일 선택
3. (선택) 필터 파라미터 입력
4. 출력 폴더 선택
5. 처리 시작

## 시스템 요구사항
- Windows 10 이상
- Python 3.9+ (빌드 버전은 필요 없음)
